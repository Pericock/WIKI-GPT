### 📖 Descripción
Este repositorio centraliza el conocimiento técnico del equipo para el grado de **ASIR**. No es solo una wiki, es una plataforma de documentación estructurada mediante **XML**, validada con **XSD** y desplegada de forma profesional.

### 📂 Estructura del Proyecto
Basándonos en nuestra arquitectura técnica, el repositorio se organiza así:

```bash
WIKI-GPT/
├── 📁 assets/      # Imágenes, diagramas y recursos visuales
├── 📁 data/        # Archivos XML con el contenido técnico
├── 📁 docs/        # Documentación final y despliegue de Pages
├── 📁 schemas/     # Archivos XSD para validación de datos
├── 📁 src/         # Código fuente y scripts de apoyo
└── 📄 README.md    # Esta guía principal
